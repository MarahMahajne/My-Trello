About Trello Web Application

Project Overview:

Trello Web Application is a self-initiated project developed from September 2023 to October 2023. It is a full-stack web application designed to enhance project management and collaboration experiences for users. The application is built using Angular for the frontend and Firebase for the backend.

Key Features:

  • Intuitive User Interface
  • Seamless Collaboration
  • Real-time Updates

Technologies Used:

  • Frontend: Angular
  • Backend: Firebase
  • Database: Firebase Realtime Database
  • Authentication: Firebase Authentication

Project Impact:

Trello Web Application aims to improve teamwork and productivity by offering efficient project management tools. With its user-friendly interface and real-time collaboration features, the application empowers users to streamline their workflows and achieve their project goals effectively.
